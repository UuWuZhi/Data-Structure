# Data Structures in C/C++

本项目使用 **C/C++** 语言实现常用基础数据结构，包括链表、队列、栈等。  
**仅供学习交流使用，请勿应用于生产环境。**

---

## 📦 核心内容

- **链表（Linked List）**：单向链表、双向链表  
- **队列（Queue）**：顺序队列、链式队列  
- **栈（Stack）**：顺序栈、链式栈  
- 更多基础结构持续扩展中……

### `CppSrc/` 与 `CSrc/` 的区别
| 特性   | CppSrc (C++)                | CSrc (C)                       |
|------|-----------------------------|--------------------------------|
| 核心语法 | 类、模板、引用、运算符重载               | 结构体、指针、函数                      |
| 内存管理 | `new` / `delete`            | `malloc` / `free`              |
| 输入输出 | `iostream` (`cin` / `cout`) | `stdio.h` (`printf` / `scanf`) |
| 数据抽象 | 通过类将数据和操作封装在一起              | 通过结构体存放数据，函数单独定义               |
| 代码复用 | 模板可生成不同类型的链表/栈/队列           | 通常用 `void*` 或宏实现泛型，较繁琐         |
- **CppSrc**：充分利用 C++ 的现代化特性，代码更简洁、类型安全，适合学习 OOP 和泛型编程。
- **CSrc**：严格使用 C 语法，适合理解数据结构的底层内存布局和指针操作，对理解操作系统、嵌入式开发有帮助。
