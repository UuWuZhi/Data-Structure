# Data Structures in C/C++

本项目使用 **C/C++** 语言实现常用基础数据结构，包括链表、队列、栈等。  
**仅供学习交流使用，请勿应用于生产环境。**

---

## 📦 核心内容

- **链表（Linked List）**：单向链表、双向链表  
- **队列（Queue）**：顺序队列、链式队列  
- **栈（Stack）**：顺序栈、链式栈  
- 更多基础结构持续扩展中……

## 📖 目录说明

### `Directions/` 文件夹
该文件夹包含若干简短的 C++ 入门指南，帮助初学者快速理解项目中所用到的 C++ 特性，例如：
- 模板的基本声明与使用
- 类的构造、析构与封装
- 引用作为函数参数
- 输入输出流（`cin` / `cout`）的简单用法
> 如果你对 C++ 尚不熟悉，建议先阅读该目录下的文档再阅读 `CppSrc` 中的代码。

### `CppSrc/` 与 `CSrc/` 的区别
| 特性   | CppSrc (C++)                | CSrc (C)                       |
|------|-----------------------------|--------------------------------|
| 核心语法 | 类、模板、引用、运算符重载               | 结构体、指针、函数                      |
| 内存管理 | `new` / `delete`            | `malloc` / `free`              |
| 输入输出 | `iostream` (`cin` / `cout`) | `stdio.h` (`printf` / `scanf`) |
| 数据抽象 | 通过类将数据和操作封装在一起              | 通过结构体存放数据，函数单独定义               |
| 代码复用 | 模板可生成不同类型的链表/栈/队列           | 通常用 `void*` 或宏实现泛型，较繁琐         |
- **CppSrc**：充分利用 C++ 的现代化特性，代码更简洁、类型安全，适合学习 OOP 和泛型编程。
- **CSrc**：严格使用 C 语法，适合理解数据结构的底层内存布局和指针操作，对理解操作系统、嵌入式开发有帮助。

## 🤝 贡献指南
> 我们欢迎任何形式的贡献！你可以通过以下方式参与本项目：

1. 提交 Issue

   - 对代码有问题的部分提出质疑：发现逻辑错误、内存泄漏或未定义行为时，请告知我们。
   - 提出疑惑：对某段实现不理解，或对数据结构的使用场景有疑问，欢迎提问。
   - 提出项目发展建议：例如希望增加哪种数据结构、改进项目结构、完善文档等。

2. 提交 Pull Request（完善现有代码）

   - 完善数据结构：尤其欢迎为 CSrc/（C 语言部分）补充缺失的数据结构（如双向链表、循环队列等），或修复现有 bug。
   - 完善注释：为关键函数、复杂逻辑添加清晰的中文或英文注释，提高代码可读性。
   - 完善现有功能：为已有数据结构增加新操作（如链表的合并、反转、查找等），或优化性能。

3. 提交 Pull Request（提供新算法）

   - 展示数据结构的应用：在 CppSrc/algorithm/ 或 CSrc/algorithm/ 中添加使用本数据结构的算法示例，例如：
   - 基于栈的表达式求值
   - 基于队列的广度优先搜索（BFS）
   - 基于链表的多项式加法
   - 其他能体现数据结构价值的经典问题

> 提交 PR 时请确保代码风格统一（缩进、命名等），并适当添加注释。如果是新算法，请附带简短的说明和测试用例。
